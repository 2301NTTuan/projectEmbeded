Mô tả:
- Phần cứng:
  + Vi điều khiển: STM32F103C8T6
  + Màn hình OLED
  + Cảm biến gia tốc MPU6050
  + Đèn LED
- Chức năng: Đọc dữ liệu từ cảm biến gia tốc để xử lý, tính toán bước chân
- Tính năng: 
  + Dừng chương trình, tiếp tục chương trình
  + Reset bộ đếm
